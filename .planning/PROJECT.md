# FlightAgent

**Project Vision:** A .NET 10 AI-powered agent that answers natural language questions about real flights using Microsoft Agent Framework and live aviation APIs.

**Why This Exists:** Travel agents and airline customer service staff currently waste time navigating multiple airline websites and learning exact query syntax to check flight status. FlightAgent lets them ask plain English questions like "Is flight KL071 delayed?" and get structured, real-time answers with proper resilience when APIs fail.

**Success Looks Like:** A minimal API endpoint that accepts natural language queries, understands intent using AI, fetches real-time flight data from aviation APIs, and returns structured answers with graceful degradation when services are unavailable.

**Target Users:**
- Travel agents needing quick flight information
- Airline customer service staff handling passenger inquiries

**Core Value Proposition:** Natural language interface to flight data eliminates manual system navigation and reduces response time for customer inquiries.
